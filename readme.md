# 项目介绍

## 目录结构
  - `oa_client`
    + 后台管理系统的客户端页面
  - `oa_server`
    + 后台管理系统的服务器
    + 开启 cors
  - `shop_client`
    + 电商网站的客户端页面
  - `shop_server`
    + 电商网站的服务器

## oa 后台管理系统
  - 对于用户的增删改查
  - 对于轮播图的增删改查
  - 对本用户信息的修改
